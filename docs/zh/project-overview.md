# 项目概览

## 定位

openCodex 是一个受 openclaw 启发的、面向代码仓库场景的 coding agent 项目。
它的目标是帮助开发者以更紧凑的本地工作流，从任务意图走到经过验证的代码改动。

## 设计目标

1. 保持工作闭环可观察、可理解。
2. 优先采用最小且可回滚的改动。
3. 将规划、执行与验证串成一个流程。
4. 贴合本地开发与命令行使用场景。

## 拟定 MVP 能力

- 将用户任务解析为结构化步骤。
- 在仓库内检索相关文件与符号。
- 生成并应用最小补丁。
- 执行聚焦型验证命令。
- 汇总改动内容与剩余风险。

## 第一版暂不覆盖

- 大规模自主重构。
- 纯云端执行模式。
- 完整替代 IDE。
