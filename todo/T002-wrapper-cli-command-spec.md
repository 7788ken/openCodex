# T002 — Wrapper CLI Command Spec

## Objective

Define the first openCodex command surface as a thin wrapper around Codex CLI.

## Scope

- Propose command names.
- Define command responsibilities.
- Define user-facing inputs and outputs.
- Keep the design small enough for a first MVP.

## Suggested Commands

- `opencodex run`
- `opencodex review`
- `opencodex doctor`
- `opencodex session`

## Acceptance Criteria

- Each command has a one-sentence purpose.
- Each command has a minimal flag set.
- The spec avoids duplicating existing Codex CLI features.
